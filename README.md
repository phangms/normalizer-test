# This is a Normalizer Testing template

There are currently 2 sample application:
1) src/app.js
2) server.js

You run them manually by:
```shell
node src/app.js
```
and
```shell
node server.js
```
respectively.

The tests are located within the `test` directory.
You can run the test by running `mocha`.
```shell
mocha
```
